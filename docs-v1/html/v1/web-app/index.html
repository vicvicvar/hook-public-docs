Web APP

La interfaz Web brinda visibilidad a todas las funcionalidades de Optacheck.


Aprende el uso y manejo de Optacheck Web APP en menos de 5 minutos.

Elije entre las siguientes opciones (recomendado segun el orden) para iniciar un tutorial:

  1. En el tutorial Basico te damos la introduccion a la herramienta y las funcionalidades mas importantes.

  2. Ingresa al tutorial Control donde exploramos Geocercas y Cumplimientos.

  3. En el tutorial Especifo , funcionalidades dedicadas como carga masiva de misiones y reporteria son demostradas

  4. O regresa al Inicio para elegir entre Web-App o nuestra App-movil
