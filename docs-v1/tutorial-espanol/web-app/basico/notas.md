# Notas

Con las notas, tus personal captura y comparte eventos importantes del viaje o mision. 

Eventos como:

 - Trafico
 - Proceso de Carga/Descarga en retraso 
 - Eventualidades 
 - Clima
 - Neumático desinflado/pinchado
 - Desperfecto mecánico
